# 前端概念

> 2005年之前，是没有「前端开发」这个名词的。在这之前，类似的岗位叫做「美工」。在前端出现之前难道就没有页面吗？有的，而且很多，只不过这些页面大多很简单，后端随便几下就写完了。直到 AJAX 出现之后，才出现一批专门以 JS 为业的程序员了，这些人就是前端。

这里的前端主要指的Web前端，随着智能终端的出现，各种终端的Native开发也被纳入大前端的范畴，不过这里我们只讨论Web相关的技术。

# Web标准

由W3C制定的Web设计及应用包括构造和渲染Web页面所需的各类技术标准，现在的Web设计为三部分组成，即结构（Structure）、表现（Presentation）、行为（Behavior），其中结构标准包括XHTML、XML，表现标准语言包括CSS，行为标准包括DOM、ECMASCRIPT。现阶段HTML标准发展至HTML5，CSS标准在CSS3，ECMASCRIPT标准已经到ES6，但是在标准的进步下，还需要浏览器的支持，所以前端的项目中经常会出现检测特性，兼容性相关的代码，并且在桌面浏览器下需要考虑IE（Edge）是否支持等头疼问题。不过随着移动端的崛起，移动端相关的标准落实较快，可以愉快的使用一些新特性，但是还是需要做部分的兼容性方案。
